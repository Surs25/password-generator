# Password Generator

A Python application that generates strong and secure random passwords.

## Features
- Generate random passwords
- Customizable password length
- Includes uppercase letters, lowercase letters, numbers, and special characters
- Helps create secure passwords for online accounts

## Technologies Used
- Python

## How to Run

```bash
python password_generator.py
```

## Example Output

Generated Password: X7@kP9!mQ2#z
